# check

check html file.

## usage

```bash
w3c-validator check <directory> [-t || --throwError]
```

> [ ] = optionnal<br/>
> < > = required

## options

| option       | alias | description                                   | required |
| ------------ | ----- | --------------------------------------------- | -------- |
| --throwError | -t    | throw error when w3c validator response error | false    |

## exemples

```bash
w3c-validator check <directory> -t // or --throwError
```
