# How to use

The cli is based by `@jokay03j-v2/w3c-validator` package to provide all commands.

## Install

<Tabs>
  <TabItem value="npm" label="npm" default>

    npm install @jokay03j-v2/w3c-validator -g

  </TabItem>
  <TabItem value="pnpm" label="pnpm">

    pnpm install @jokay03j-v2/w3c-validator -g

  </TabItem>
  <TabItem value="yarn" label="yarn">

    yarn global add @jokay03j-v2/w3c-validator

  </TabItem>
</Tabs>

## Use it !

You can use `w3c-validator` inside your terminal !<br/>
To learn all commands you must go [here](./commands/check.md)
