# W3C Validator

> W3C validator is a cli and provide apis to check html file

## CLI

### How to use ?

#### Install package

> You can install globaly but is not recommended

```
npm i @jokay03j-v2/w3c-validator
```

#### Run cli

```
npx @jokay03j-v2/w3c-validator check <path to html file> -t
```
